# Cohort Analysis — Customer Retention

Analysis of customer retention patterns using SQL (DuckDB) and Python.

**Dataset:** Product Sales Dataset 2023-2024 (Kaggle)  
**Tools:** Python, DuckDB, Pandas, Seaborn  
**Key finding:** Average M+1 retention is 4-5%; strong seasonal spike in Oct/Nov 
suggests Black Friday effect across all cohorts.
